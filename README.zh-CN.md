# 静痕工作台

静痕工作台是一组安静、克制、专注的思源笔记实用工具。

## 当前工具

右上角“静痕工作台”集中展示所有工具及当前状态，并提供每日笔记、文档树定位、字体和 PDF 导出的入口。文档树自动定位也可以直接在工作台中开关；从工作台进入设置后可以直接返回，保存成功也会回到工作台。

### 文档内查找

在文档正文中按 `Command + F`（Windows/Linux 为 `Ctrl + F`），会在当前编辑器右上角打开紧凑查找条。输入内容后，插件仅搜索当前文档，按正文顺序高亮并跳转匹配项；`Enter` 前往下一项，`Shift + Enter` 返回上一项，`Esc` 关闭。

思源原生 `Command + P` 全局搜索保持不变。插件不会在工作台、设置、PDF 或其他非文档区域接管 `Command + F`。

### 每日笔记

左上角日历按钮提供两个入口：

* 左键点击可创建或打开今天的每日笔记。
* 桌面端右键点击、移动端长按，可查看今天及前 6 天的每日笔记；未创建的日期只显示状态，不会补建历史笔记。

在插件设置中选择目标笔记本、日记主目录和可选模板。主目录例如 `/日记`，实际每日笔记路径固定为 `主目录/年/月/YYYY-MM-DD`。

模板支持两种来源，二选一：

* 打开任意文档页面，在设置中点击“使用当前页”；插件会在首次创建当天日记前刷新模板快照。
* 填写 `data/templates` 下的相对模板路径，例如 `daily.md`。

还可以开启“启动时自动创建”。打开思源后，插件会静默确保当天日记存在，但不会自动打开页面。路径和模板继续复用所选笔记本的思源原生每日笔记配置。

### 文档树定位

日历按钮右侧提供手动定位按钮，可在左侧文档树中展开并选中当前笔记。插件设置中的“文档树定位”可以开启自动定位；也可以在桌面端右键定位按钮快速开关。定位按钮不提供长按菜单。

### 快速更换字体

左上角提供字体与字号按钮。菜单顶部可以按 1px 增减字号、输入具体字号或重置为 16px；下方可搜索系统已安装字体。鼠标悬停在字体项上时，正文会即时临时预览；关闭浮动菜单会恢复原字体，只有点击字体项才会保存，也可恢复默认字体。

### 精美 PDF 导出

左上角 PDF 按钮会读取当前文档并打开独立预览。导出默认跟随当前编辑器字体，也可以为本次导出选择系统已安装字体；默认根据 H1-H3 标题生成 PDF 阅读器左侧的书签大纲，不在正文中插入目录页。还可选择“清阅”“极简”或“报告”版式，以及 A4、A5、Letter 页面大小和舒展、紧凑两档留白，并决定是否包含子文档或保留折叠状态。

导出沿用思源导出预览 API，保留图片、表格、代码、数学公式和常见渲染块，并针对标题层级、长表格、代码块和分页进行 PDF 排版。点击“导出 PDF”后选择本地目录和文件名，文件会直接写入，不再打开打印机或系统打印面板。

### 语义块样式

从内容块菜单进入“静痕工作台”，可以为一个或多个内容块设置以下语义角色：

* 笔记
* 提示
* 警告
* 重要
* 弱化

插件使用 `custom-stillmark-role` 块属性保存角色，并提供克制的亮暗模式基础样式。清除角色不会修改块正文。

## 开发

需要 Node.js 24 或更高版本以及 pnpm 11。

```bash
pnpm install
pnpm dev
```

生产构建与完整检查：

```bash
pnpm check
```

构建产物位于 `dist/`，集市安装包为 `package.zip`。

## 安全边界

* 笔记和块数据只通过思源内核 API 读写。
* 不直接读写工作空间的 `data` 目录。
* 文档内查找只通过思源搜索 API 读取当前文档，不修改块正文。
* 每日笔记只会写入用户明确配置且当前已打开的笔记本。
* 最近 7 天菜单只读，不会创建缺失的历史日记。
* 批量操作仅作用于用户明确选择的内容块。
* PDF 导出只读取当前文档及用户明确选择包含的子文档，不修改笔记内容。
